# Website Rental PlayStation (GameZone)

Website ini adalah platform modern untuk layanan penyewaan konsol game (PS4, PS5) yang dibangun menggunakan teknologi web terkini. Website ini dirancang untuk memberikan pengalaman pengguna yang seamless, mulai dari melihat katalog, mengecek detail spesifikasi, hingga melakukan pemesanan langsung via WhatsApp.

## ğŸš€ Fitur Utama

- **Katalog Produk Interaktif**: Menampilkan daftar konsol PS4 dan PS5 dengan harga harian, mingguan, dan fitur filter kategori.
- **Halaman Detail Lengkap**: Informasi mendetail mengenai spesifikasi konsol, game yang terinstall, dan kelengkapan paket.
- **Kalkulator Harga Otomatis**: Menghitung estimasi total biaya sewa berdasarkan durasi hari dan tambahan aksesoris (stick, headset, monitor).
- **Integrasi WhatsApp Pintar**: Pemesanan langsung terhubung ke WhatsApp Admin dengan format pesan yang sudah terisi otomatis (Pre-filled message) berisi detail pesanan.
- **Desain Responsif & Premium**: Tampilan visual yang menarik dengan animasi halus, dark mode aesthetic, dan layout yang optimal untuk Mobile maupun Desktop.
- **Informasi Lokasi & Kontak**: Integrasi Google Maps dan informasi kontak yang mudah diakses.
- **Fitur Scroll-to-Top**: Navigasi yang mudah bagi pengguna.

## ğŸ’» Cara Instalasi & Menjalankan

Ikuti langkah-langkah berikut untuk menjalankan project ini di komputer lokal Anda:

1.  **Clone Repository**

    ```bash
    git clone https://github.com/rivanalamsyah/playstation-rental-website.git
    cd playstation-rental-website
    ```

2.  **Install Dependencies**
    Pastikan Node.js sudah terinstall.

    ```bash
    npm install
    ```

3.  **Jalankan Development Server**

    ```bash
    npm run dev
    ```

    Website akan berjalan di `http://localhost:5173`.

4.  **Build untuk Production**
    ```bash
    npm run build
    ```
